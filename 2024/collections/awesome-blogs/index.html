Awesome blogs | 优质 Blog 收集

发表于2024-12-22|更新于2025-12-23|Collections
|浏览量:

一直以来都是自己私藏一些 Blog, 现在整理一下自己收集的 Blog, 作为分享, 也作为 backup

Official

一些互联网企业官方的 Blog 包含

  • 美团技术团队: 有很多美团内部遇到的技术问题和技术分享, 有一些比较有价值的分享; 但是最大的问题就是由每个技术团队自己撰写, 水平参差不齐, 有的连排版都做不好, 或者是非常浅显的介绍分析.
  • 有赞技术团队: 似乎是一个新兴的电商平台的博客.
  • 得物技术官网
  • 公众号: 腾讯技术工程

Personal

  • 星合の空: 中大博士, 一些计算机体系结构的分享, 作者绝对大神, 但是最近更新频率太低
  • TianYu GUO’s homepage: 一些体系结构的硬核文章
  • Evian Zhang’s naive blog: 中科大博士, 记得当初是看了作者的 llvm ir 的教程才发现的 blog, 但是似乎首页放了一些 macos 汇编和 latex 相关的东西
  • 谭邵杰的计算机奇妙之旅: 一些体系结构相关的
  • 4G Space: 徐宥, 之前在 Fitbit (领导机器学习团队) 和 Google (广告和无人驾驶) 干活, 华盛顿大学人工智能博士, 南京大学数学学士. 量化相关的文章
  • Yesterday17’s Blog: 似乎主要是做一些逆向相关的
  • decodez: 一些 CPU 尤其是缓存相关的文章
  • 阮一峰的网络日志: 相信很多人学习时候都看到过的优质博客
  • 孙同学博客: 主要是一些网络协议详解, 也有一些其他的技术分享.
  • 三月沙: 主要是一些 golang, web 后端的技术分享
  • StormQ’s Blog: 一些编译, 性能相关的
  • Lequn Chen || abcdabcd987: 体系结构, 高性能相关的
  • 博客园 - 返回主页大奥特曼打小怪兽: 一些嵌入式的驱动相关的开发博客
  • tsecer: 一些底层的技术知识, 基本上保持月更
  • 加琪的技术记事本: 深度学习, 后端前端, 各种都会涉及, 更新频率较高

面筋

来吃一串面筋吧!

C++ 的没有看到过特别好的系统的面经, 算法 (机器学习), 体系结构等也没有比较系统的

  • 小林 x 图解计算机基础: 很多应届生的第一本面经, 但是现在看起来是最好的校招生免费面经了
  • JavaGuide: 还不错的 Java 面筋
  • Java 全栈知识体系: 各种东西都有, 但是还没有看过
  • 前端面试题集锦: 前端面试题

算法题

大部分都是比较入门的, leetcode 难度不是特别大

  • Leetcode top 100 liked: 很多人都刷完了
  • 代码随想录: 算法面经
  • 算法通关手册（LeetCode）: 算法面经, 题目比较多
  • OI wiki: OI wiki, 更多的面向竞赛
