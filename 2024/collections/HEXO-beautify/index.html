SEO

首先就是 Google Search Console 里面配置一下站点, 因为这个 zombie12138.github.io 的这个 domain 不是我们的, 我们选择网址前缀, 输入之后只要把 google 的这个文件验证放到仓库的根目录, 然后点击验证就可以啦.

但是这样似乎还是不够的, 等了一天也没什么动静, 指定 site 搜索了一下还是没有什么输出. 顺便把 Sitemap 配置一下.

参考文献脚注

所以网上找了一下, 首先看到的是 hexo-reference, 但是看了一个 例子, 感觉没有悬浮效果不太方便, 又看了一篇 博客, 他是自己配置的 Hint.css.

同步

Hexo 的 deploy 命令（如 hexo d）默认只负责部署 public 目录中生成的静态网站文件. 对于博客的 Markdown 源文件, 主题配置等, 如果需要同步, 则需要我们手动通过 Git 等工具进行版本控制.

已经自带了 .gitignore, 可以直接按照普通的项目在根目录进行 git 操作就可以了
