TVM入门 -- 0 Hello World

发表于 2024-05-03

来系统看一下TVM的教程, 本文将包含安装和一些基本的使用, 基本上参考照抄tvm官方文档和中文文档

其实是为了熟悉一下怎么使用, 之前用过, 生成的模型在BatchSize大的时候很慢

简介

TVM是一个深度学习编译器, 但是很怀疑有什么用, 因为之前生成Kernel时候, batch size大的时候, 性能总是很差, 稍微学习一下到底是什么东西.

其步骤原理就是:

  1. 从TF, Torch, 或者ONNX的模型, 导入
  2. 把模型翻译成Relay IR, 会加入一些图优化
  3. 降级为TE (Tensor Expression), FuseOps pass会把模型切分为小图.
  4. 使用auto-tuning模块AutoTVM或AutoScheduler搜索最佳schedule进行一些循环优化(切分, 融合, 展开等), 并行化之类的优化
  5. 生成JSON的调优记录, 为每个子图选择schedule
  6. 降级为TensorIR, 进一步优化(降级到硬件)
  7. 生成机器码

Install

跟着教程就行了

注意Cmake要3.20以上. 我开启了CUDA, NCCL, NVTX, OpenCL, OpenMP, RELAY_DEBUG和LLVM, 一些库没有就可以在CMakeList.txt里面添加一些link_directories或者include_directories来解决

最终效果就是python里面能import tvm, 并且能执行python -m tvm.driver.tvmc

比较坑的是llvm版本, 官网只说了4.0以上, 但是17不可以, 下面的时候会Segment Fault, 14就可以… 搜了一下, 官网没有进行其他额外的说明, 而且也没有搜到什么要求.
