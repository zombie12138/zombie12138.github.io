HEXO 美化

发表于 2024-08-10 更新于 2025-03-29

这个 Blog 只安装了一个 next 主题, 就再也没有管过, 来浅浅的折腾一下. 持续更新~

SEO

首先就是 Google Search Console 里面配置一下站点, 因为这个 zombie12138.github.io 的这个 domain 不是我们的, 我们选择网址前缀, 输入之后只要把 google 的这个文件验证放到仓库的根目录, 然后点击验证就可以啦.

但是这样似乎还是不够的, 等了一天也没什么动静, 指定 site 搜索了一下还是没有什么输出. 顺便把 Sitemap 配置一下.

首先在 next 里面把 themes/next/_config.yml 修改一下, 打开注释

安装插件

直接 hexo g, 这时候 public 文件夹多了 sitemap.xml 和 sitemap.txt, hexo d, 在 Google Search Console 里面提交这个 xml 的 sitemap 就可以了
