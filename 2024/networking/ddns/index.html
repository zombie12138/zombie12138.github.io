使用 DDNS | ddclient

发表于2024-04-23|更新于2025-12-23|Networking
|浏览量:

家里面的宽带竟然是有公网 IP 的, 想要利用这个公网 IP, 于是家里放了一个玩客云的盒子 (ARMv7), 现在来给这个 IP 加一个 DDNS, 方便以后跑 frp 和 wireguard

dynv6

首先没有 DNS, 选择了一个免费的 ddns 网站 dynv6, 注册之后直接添加 zone, 然后填入自己当前的 IP 试试, 得到 zombie12138.dns.army 这样的域名

build

就是使用 ddclient, 定期汇报自己的 IP

机器本身是 32bit arm, 并且本身也没有 release 的二进制, clone 代码然后编译

之后就被安装到了上面配置的这些个目录里面了.

配置和安装

使用 ip.sb 这个 api 来获取 ip, 300s 上报一次.

之后路由器上配置好 dmz, 让这个 OneCloud 小主机做反代 & Wireguard 的服务器端口直接暴露外网即可.

换成 docker

后面还是换成 docker 了, 毕竟 docker 在部署和管理上更加便捷, 不过下面的这个镜像不支持 armhf

直接 docker-compose up -d 运行, 然后修改配置重启即可
